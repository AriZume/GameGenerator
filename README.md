# 🎲 Game Generator (Java Console Board Game)

A customizable board game built in **Java** that runs in the console.
Players roll dice, move across tiles, trigger events, collect points, and compete under different winning rules.

This team project focuses on **clean architecture**, **object‑oriented design**, and **extensibility**, making the code easy to understand and expand.

## ▶️ How to Run

### Requirements

* Java 8 or newer
* Any Java IDE (IntelliJ IDEA recommended)

### Running the Project

1. Clone the repository
2. Open it in your IDE
3. Locate the file:

   ```
   src/Main/Main.java
   ```
4. Run the `main` method to start the game

---

## 🕹️ Gameplay Overview

1. Start a new game, quick game, or load a saved game
2. Enter player names
3. Roll dice to determine turn order
4. Players take turns:

   * Roll dice
   * Move across the board
   * Trigger tile effects
   * Apply card effects (if any)
5. The game ends when a winning condition is met
